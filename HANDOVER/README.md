# UniformHub v1 MVP

UniformHub is a community-driven marketplace for pre-loved school uniforms and equipment. This MVP is built with Next.js, Firebase, and Tailwind CSS.

## Core Features
- **Listing Management**: Users can create, edit, and manage listings with quantity support (1-3 items).
- **Search & Discovery**: Filter by school, category, size, and condition.
- **Messaging**: Real-time communication between buyers and sellers.
- **Admin Panel**: Dedicated dashboard for managing users, listings, and reports.
- **Safety**: AI-powered condition audits and community reporting.

## Technology Stack
- **Frontend**: Next.js 14+ (App Router), Tailwind CSS, Framer Motion.
- **Backend**: Firebase Auth, Firestore, Firebase Storage.
- **Infrastructure**: Firebase Hosting.
- **Extensions**: Firebase Trigger Email (for reports/notifications).

## Current Status
- **Phase**: MVP Production Handover (28 April 2026).
- **Backend**: Connected to `sniperform-ads-dashboard` (Production).

## Getting Started
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Set up environment variables (see `HANDOVER/ENV.md`).
4. Run locally: `npm run dev`.
